# Data Landing Page

A React + Vite landing page showcasing worldwide data coverage with an interactive map.

## 🚀 Quick Deploy to Vercel

### Repository: https://github.com/HBG20/landing-data.git

1. Go to [vercel.com](https://vercel.com) and sign up/login
2. Click "Add New..." → "Project"
3. Import repository: `HBG20/landing-data`
4. Click "Deploy" (settings are auto-detected)
5. Your site will be live in 1-2 minutes!

## 📦 Build

```bash
npm install
npm run build
```

## 🛠️ Development

```bash
npm run dev
```

## 📁 Project Structure

- `src/` - Source code
- `src/components/` - React components
- `src/i18n/` - Internationalization (English, Spanish, Italian)
- `src/countriesData.json` - Country data
- `vercel.json` - Vercel deployment configuration

## 🌍 Features

- Interactive world map
- Country data filtering
- Multi-language support (EN, ES, IT)
- Responsive design
- Market type legend
- Search functionality
